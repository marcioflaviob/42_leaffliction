1. SCAN   — Walk subdirectories, collect (image_path, label) pairs
2. ENCODE — Map label strings → integers, save classes.json
3. AUGMENT — For each image generate N variants:
               - horizontal flip
               - random rotation (±30°)
               - brightness/contrast jitter
               - random zoom crop
             Save augmented images to a temp folder
4. LOAD   — Load all originals + augmented into arrays, normalize to [0,1]
5. SPLIT  — Train / Validation split (e.g. 80/20)
6. BUILD  — Load MobileNetV2(weights='imagenet', include_top=False)
             Freeze base, add GlobalAveragePooling + Dense(num_classes, softmax)
7. TRAIN  — model.fit() with early stopping
8. SAVE   — Save model.h5 + classes.json + augmented images → output.zip
